# 금일 복습 내용 : 헷갈릴 수 있는 내용 정리
------------------------------------<br>
# 정적 할당 vs 동적 할당 (new / delete)
메모리를 언제, 얼마나 확보하느냐의 차이.<br>
정적 할당: 컴파일 시 크기가 결정됩니다 (예: int arr[10]; 아주 평범한 변수 메모리). 함수가 끝나면 자동으로 해제됩니다.<br>
동적 할당: 실행 중(runtime)에 필요한 만큼 메모리를 가져옵니다 (new 키워드 사용). 사용 후 반드시 delete로 직접 해제해주지 않으면 메모리 누수가 발생합니다 <br>

# 구조체(struct) vs 클래스(class)
C++에서 두 개념의 기술적 큰 차이<br>
구조체: 기본 접근 제어자가 public입니다 (외부에서 자유롭게 접근 가능).<br>
클래스: 기본 접근 제어자가 private입니다 (외부 접근 차단, 캡슐화 강조).<br>

# 입력 함수 (cin vs getline)
문자열을 입력받을 때 공백 처리 방식 때문에 자주 헷갈립니다.<br>
cin >> str: 공백(스페이스, 탭, 엔터)을 만나면 입력을 멈춥니다.<br>
getline(cin, str): 엔터(\n)를 치기 전까지의 공백을 포함한 전체 한 줄을 입력받습니다.<br>

# 얕은 복사 vs 깊은 복사 
객체를 복사할 때 발생하며, 특히 포인터 멤버가 있을 때 중요<br>
얕은 복사: 주소값만 복사합니다. 한쪽에서 데이터를 지우면 다른 쪽도 망가지는 문제가 생깁니다.<br>
깊은 복사: 실제 데이터 내용을 별도의 메모리에 새로 복사하여 두 객체를 독립적으로 만듭니다.<br>
