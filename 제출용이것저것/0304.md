break 문은 잘 사용하자. 잘못하면 return과 같은 역할을 한다. 

```c++
main(){
  while (1) {
```

if문장에서 이거이거가 참일때만 이거이거를 확인하고싶다라는걸 만들려고 했다.
과제를 하다가 3번과 4번을 고른 후 HP와 MP변수 두 수에 값이 존재하면이라는 조건의 if문을 짜려고 했는데 생각보다 과제가 어렵다 생각했다.
근데 그냥 내가 잘못 읽은 거였다.
HP와 MP 변수를 하나씩 넣는건 줄 알았는데 동시에 넣는거였다. 그러면 이런 검사식은 필요자체가 없었는데.
```c++
if ((i == 3 || i == 4) && (j[1] == 0 || j[2] == 0)) {// 입력검사!
```
해당 코드식이 내가 만든 코드이다. &&는 그리고라는 의미. 둘다 참일때를 의미한다.
j[1] 과 j[2]는 각각 hp포션,mp포션에 값이 들어갔을때 1이 들어가는 임의로 내가 추가한 배열이다. 입력검사를 할때 쓰려고 만들었다.
