# 3번과제를 위한 발버둥.
1번과 2번 과제에 대해 이렇다할 TIL이 없던이유는 알던 내용들이였기 때문.<br> 
혹은 개념을 이해하고 있으나 정확한 사용이나 응용이 부족해 자료를 조금 들여다 보는 수고가 필요했던 경우였다.<br>
허나 3번 내용은 템플릿과 vector을 주로 사용할 것으로 예상된다. 이 부분은 개념은 이해하고 있으나 어떻게 생겼는지조차 기억이 안나기에 적어간다.<br>
뭔가 강의자료가 있고 개념에 대해서 알려주니 그걸 보면서 하다보면 너무 재미없다. 답지를 보면서 답을 푸는 느낌이다. 그래서 안보고 일단 구현해보고 빨간줄일때 개념을 읽자.<br>

# 첫번째 난항 생존영어가 필요하다
```c++
templayte(typename T){} /// <- 굉장히 틀려먹음 
class Inventory {
public: // 클래스는 기본적으로 private이기 때문에 클래스 만듬과 동시에 public 존을 만들어두는 편이다

private:

};

```
첫번째 템플릿에는 template이 옳은 철자이다. 두번째 자료형 이름을 정할땐 <>안에 쓴다. ()은 안쓴다. 세번째 {}는 애초에 안쓴다. 자료형 이름을 이걸 쓰겠다고 선언하는거로 끝이기 때문.<br>
템플릿에 대한 이해도를 높여주는 좋은 오답들이였다.<br>
